# Raise Me BeOS

Next.js portfolio and donation app with a separate Hono payment service.

## Start here

- `docs/AGENTS.md` — how AI agents should work in this repo.
- `docs/README.md` — documentation index.
- `docs/architecture.md` — project structure and runtime flow.
- `docs/setup.md` — setup, scripts, and environment values.
- `docs/conventions.md` — code and UI conventions.
- `docs/quality.md` — verification rules and cleanup notes.
- `docs/DESIGN.md` — visual design system.

## Quick run

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run verify`

## Project areas

- `app/` — front-end routes and app services.
- `components/` — shared layout and UI primitives.
- `lib/` — shared helpers and config.
- `skeleton-data/` — copy and content models.
- `server/` — Hono payment service.
- `messages/` — locale bundles.

## Support

Donations remain part of the project, but the docs above are the working entry point for contributors and agents.
